# 🗓 Event Countdown Task Generator

Generate a complete reverse-timeline task plan from T-16 weeks to post-event. Edit, check off, and export tasks as you go — or import a saved template.

**Works on Mac · Windows · Linux · No install required**

---

## Quick Start

1. [**Download**](https://github.com/Visual-Hive/vh-conference-toolkit/releases/latest) `event-countdown-planner.zip` from the releases page
2. Unzip it
3. Double-click `index.html`
4. Enter your event name, date, and type — click **Generate Plan**

That's it. No server, no Docker, no setup.

---

## Saving your work

Your plan is saved automatically in your browser (localStorage). To avoid losing it:

- Use **Export as Template** (📤) to save a `.json` file to your computer
- Use **CSV Export** (⬇) for a spreadsheet-friendly backup
- Import the `.json` template any time to restore your plan

---

## Features

- **Auto-generated plans** for 4 event types: conference, gala/awards, exhibition, hybrid
- **Week-by-week view** from T-16 weeks to post-event
- **Slide-out panel** to edit titles, add notes, assign responsibility, set next actions
- **Progress tracking** — check tasks off as you go
- **Template export/import** — save your customised plan and reuse it for future events
- **Push to Ops Tracker** — scan for a local Ops Tracker and send all tasks to it in one click
- **CSV export** for spreadsheet use
- **Print-friendly** layout

---

## Connecting to Ops Tracker

If you also have the **Event Ops Tracker** installed (Option B Docker install), you can push all generated tasks directly into it:

1. Make sure the Ops Tracker is running at `http://localhost:8080`
2. Click **Send to Ops Tracker** in the Countdown Planner
3. It will detect the running instance and push all tasks to Draft
